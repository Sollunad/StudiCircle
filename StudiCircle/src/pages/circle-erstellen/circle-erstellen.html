Name* Standort Sichtbarkeit öffentlich privat

Alle mit * Markierten Felder müssen ausgefüllt werden

Name Hier können sie einen beliebigen Namen für den Circle angeben unter dem dieser dann angezeigt wird

Standort Hier können sie einen beliebigen Standort für den Circle angeben um Leute in der Umgebung darauf aufmerksam zu machen.

Sichtbarkeit Hier können sie auswählen ob alle User ihren Circel sehen und ihm beitreten können (Öffentlich), oder ob das nur auf Einladung möglich sein soll (Privat)
